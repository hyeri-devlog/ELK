## 요약

- **상품 검색**: 쇼핑몰에서 상품명, 브랜드, 카테고리 등을 빠르게 검색
- **게시글/문서 검색**: 제목, 본문, 작성자 기준으로 게시글이나 문서 검색
- **자동완성**: 검색어 일부만 입력해도 추천 검색어 제공
- **로그 수집/분석**: 서버 로그, 에러 로그, API 로그를 한곳에 모아 검색
- **모니터링**: API 응답 시간, 에러 발생량, 트래픽 변화 확인
- **장애 분석**: 특정 시간대의 에러 로그, 느린 API, 실패 요청 추적
- **보안 분석**: 로그인 실패, 비정상 접근, 반복 요청 등 탐지
- **데이터 집계**: 카테고리별 상품 수, 시간대별 주문 수, 에러 코드별 발생 횟수 분석
- **AI 검색**: 벡터 검색, 의미 기반 검색, RAG 검색에 활용
